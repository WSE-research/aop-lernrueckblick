# AOP PVL2 — Persönliche Lernrückblicke (verschlüsselt)

Statische, **client-seitig mit StatiCrypt (AES) verschlüsselte** Einzelseiten für die
Teilnehmenden der AOP-PVL2-Pilotstudie (HTWK Leipzig, WSE Research, Mai 2026).

- Pro Teilnehmer:in eine Seite unter `u/<id>/`, nur mit individuellem Passwort lesbar.
- Es liegen ausschließlich **pseudonymisierte** Inhalte vor (Teilnehmer-ID, keine Namen).
- Quelldaten/Analyse liegen NICHT hier (privates Analyse-Repo).

Gehostet über GitHub Pages.
